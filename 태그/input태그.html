사용목적
form태그 내에서 입력 형식을 지정하기위해 사용하는 태그



관련 속성
type=“text” : 한줄 텍스트 입력창을 만들고 싶을 때 사용하는 속성

type=“password” : 비밀번호 입력창을 만들고 싶을 때 사용하는 속성

-value=“ ” : 초기에 출력하는 문자열을 배정하고싶을 때 사용하는 속성

-maxlength=“ ” : 입력할 수 있는 최대 문자개수를 설정하고싶을 때 사용하는 속성

-size=“ ” : 입력창의 가로크기를 설정하고싶을 때 사용하는 속성

list=“ ” : type=“text”와 함께 사용되며 드롭다운 버튼(▼)을 이용해서 datalist태그의 리스트를 출력하기 위해 사용하는 속성


type=“button” : 아무기능없는 단순버튼을 만들기 위해 사용하는 속성

type=“submit” : 폼데이터를 웹 응용프로그램에 전송하는 버튼을 만들기 위해 사용하는 속성

type=“reset” : 폼에 입력한 데이터를 초기화하는 버튼을 만들기 위해 사용하는 속성

type=“image” : 이미지 버튼을 만들기 위해 사용하는 속성(추가적으로 이미지 속성을 이용)

-value=“ ” : 클릭버튼에 출력할 텍스트를 만들기 위해 사용하는 속성


type=“checkbox” : 여러 가지 선택가능한 박스를 만들기 위해서 사용하는 속성

type=“radio” : 한 개만 선택가능한 박스를 만들기 위해서 사용하는 속성

-value=“ ” : 요소값을 설정할 때 사용하는 속성

-name=“ ” : 요소이름을 설정할 때 사용하는 속성

-checked : 초기에 선택된 상태로 출력하기 위해서 사용하는 속성


type=“color” : 색을 입력하기 위해서 사용하는 속성

-value=“ ” : 색의 값을 지정하기 위해서 사용하는 속성(#rrggbb)


type=“month” : 년/월 정보를 입력하기 위해서 사용하는 속성

type=“week” : 년/몇 번째 주 정보를 입력하기 위해서 사용하는 속성

type=“date” : 년/월/일 정보를 입력하기 위해서 사용하는 속성

type=“time” : 시/분 정보를 입력하기 위해서 사용하는 속성

-value=“ ” : 시간 정보의 값을 지정하기 위해 사용하는 속성


type=“number” : 스핀버튼(⧗)을 이용해서 숫자를 입력하고 싶을때 사용하는 속성

type=“range” : 슬라이드바를 이용해서 숫자를 입력하고 싶을 때 사용최솟값을 설정하고 싶을 때 사용하는 속성

-min=“ ” : 최솟값을 설정하고 싶을 때 사용하는 속성

-max=“ ” : 최댓값을 설정하고 싶을 때 사용하는 속성

-step=“ ” : 스핀버튼(⧗)을 한번 클릭했을 때 이동하는 값을 설정하기 위해 사용


type=“email” : email형식만 인식하는 텍스트를 입력하기 위해 사용하는 속성

type=“url” : url형식만 인식하는 텍스트를 입력하기 위해 사용하는 속성

type=“tel” : 전화번호형식만 인식하는 텍스트를 입력하기 위해사용하는 속성

type=“search” : 검색어 입력 창을 만들기 위해 사용하는 속성

-placeholder=“ ” : 형식을 지정하기 위해 사용하는 속성(“id@host” “http://” “010-1234-5678” ＂검색어“)
