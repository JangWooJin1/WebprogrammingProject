자식 셀렉터
포함되는 태그(자식 태그)를 구체적으로 지정하는 것으로 자식 셀럭터는 바로 밑단계의 자식 태그에만 적용
div > div > strong {color : blue;}



자손 셀렉터
포함되는 태그(자식 태그)의 자식 태그까지 모두 지정하는 것

div div strong{color : blue;}



전체 셀렉터
와일드카드 문자(*)를 사용하여 웹 페이지의 모든 HTML태그에 적용할 스타일을 만드는 셀렉터

*{color : blue;}



속성 셀렉터
HTMl태그의 특정 속성에 대한 값이 동일한 태그에만 스타일 적용

input[type=text]{color : blue;}



가상 클래스 셀렉터
특정한 이벤트가 발생하였을 때 적용하도록 만들어진 셀렉터

a:hover{마우스가 올라갈 때 적용}
a:active{마우스로 누르고 있는 상황에서 적용}
form:focus{폼 요소가 키보드나 마우스 클릭으로 포커스를 받을 때 적용}
