태그 이름 셀렉터
태그 이름을 셀렉터로 사용하는 것
h2, li{color : blue;}



class 셀렉터
class 속성으로 셀렉터를 지정하는 것으로 점(.)을 이용

body.class{color : blue;}



id 셀렉터
id 속성으로 셀렉터를 지정하는 것으로 샾(#)을 이용

body#class{color : blue;}
